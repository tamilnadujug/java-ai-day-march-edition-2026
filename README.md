# Java – AI Day (March Edition) | Coimbatore

## About Event

The TamilNadu Java Users Group (TNJUG) brings you a special **Java – AI Day (March Edition)** — a full-day community event dedicated to exploring how Java developers can build real AI-powered applications.

This is not just a single workshop. It’s a complete AI learning day combining:

- **A deep hands-on Spring AI workshop**
- **Multiple AI tech talks & discussions**
- **Real-world architecture insights**
- **Community interaction with industry engineers**

Hosted at **EPAM Systems, Coimbatore**, this event focuses on practical adoption of AI inside modern Java systems — not theory, not hype, but actual implementation.

## Event Details

- **Date**: 7 March 2026  
- **Time**: 9:00 AM – 5:00 PM  
- **Venue**: EPAM Systems, Coimbatore

## Main Workshop — Hands-on Spring AI

This is **NOT** a beginner session. You will build, integrate, and understand how AI works inside production-style applications.

### What you will learn

- **Spring AI architecture and core concepts**
- **Building AI features using Spring Boot**
- **Prompt engineering from backend applications**
- **RAG & data integration patterns**
- **Connecting databases and enterprise systems with LLMs**
- **Handling real production constraints**
- **Debugging and optimizing AI responses**

⚠️ **Bring your laptop — the entire session is coding-focused.**

## AI Tech Talks & Meetups

Alongside the workshop, we’ll have focused discussions and sessions around:

- **Designing AI-enabled enterprise applications**
- **When NOT to use AI in backend systems**
- **Performance and cost considerations**
- **Real experiences from production projects**
- **Open Q&A with speakers and community engineers**
